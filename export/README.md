# Vet3 Equine Care - Export Package

This package contains everything you need to rebuild the Vet3 Equine Care website in WordPress with Kadence theme and Kadence blocks.

## Package Contents

### 1. **WIREFRAMES.md**
Complete visual wireframes for all 8 pages with Mermaid diagrams showing layout structure and component hierarchy.

### 2. **STYLE-GUIDE.md**
Comprehensive visual style guide including:
- Complete color palette (HSL, Hex, RGB values)
- Typography system with exact sizes
- Spacing system and grid layouts
- Button variants and states
- Form elements styling
- Card components
- Navigation and footer styles
- All design tokens needed for Kadence

### 3. **css/styles.css**
Fully functional, production-ready CSS with:
- Cross-browser compatible styles
- Responsive design for all breakpoints
- All components styled and ready
- CSS custom properties for easy customization
- Print styles included

### 4. **js/main.js**
Complete JavaScript functionality:
- Mobile navigation with hamburger menu
- Accordion functionality for services
- Form handling and validation
- Toast notifications
- Smooth scrolling
- Active nav state
- Lazy loading images

### 5. **HTML Pages** (in html/ folder)
All 8 pages as fully functional HTML:
- index.html (Home)
- about.html
- services.html
- register.html
- contact.html
- zone-visit.html
- terms.html
- privacy.html

## How to Use with WordPress/Kadence

1. **Import Design Tokens**: Use the STYLE-GUIDE.md to configure Kadence theme colors, typography, and spacing
2. **Reference Wireframes**: Follow WIREFRAMES.md for exact layout structure
3. **Copy Components**: Use the HTML as reference for building Kadence blocks
4. **Styling**: The CSS file provides all styling rules that can be added to Kadence custom CSS
5. **Functionality**: The JavaScript can be added to WordPress footer or as a custom script

## Quick Start

1. Open WIREFRAMES.md to understand page structure
2. Review STYLE-GUIDE.md for all design specifications
3. Open HTML files in browser to see fully functional site
4. Use styles.css and main.js as-is or customize as needed
